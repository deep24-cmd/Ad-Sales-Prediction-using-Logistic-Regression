# Ad-Sales-Prediction-using-Logistic-Regression

Advertisement Sale Prediction using Logistic Regression
**Project Description**

This project predicts whether an existing customer will purchase a product after seeing an advertisement, based on their Age and Estimated Salary.
The prediction is made using a Logistic Regression machine learning model, which is well-suited for binary classification problems.

Data loading
